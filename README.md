# Pre Configured Workflow usage
- - As mentioned earlier anyone can use this with no gurantee
- - How to add a mod?-
  - Get the mod link from any platform(git,gitlab,etc)
  - Copy-paste the link in `/mods.txt`
  - Keep adding mods witht the restriction of one mod per line
  - After inserting the mod/mods go to actions-> workflow and run bulk add and release
  - After initiating the workflow download the new release

    # Important
    - If you're attempting any changes you'll have to delete the current worldmods folder on your server in this case Techblox
